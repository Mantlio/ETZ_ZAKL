Данное ЕТЗ участника ВсОШа по информатике в профиле "Робототехника" для выполнения задания

Это устройство ездит по рейкам и может рисовать на бумаге А4

Навигация по файлам:
    Принципиальная Схема (Э3) - схема соединеных компонентов
    Структурная Схема (Э1) - краткая сводка как комутируют компоненты
    PCB_плата - файла с Gerber файлом для её рассмотра или заказа
    Модели - 3D модели в формате m3d/a3d для САПР Компас 3D или step для производства:
        Интерфейс - модель держателя для OLED дисплея и инкрементального энкодера
        Колесо(шестерня) - Модель шестерни для езды по рейкам
        Мотор - модель шагового мотора и пару конструктивов для его крепления
        Основание - основание самого ЕТЗ
        ось_z(маркер) - пару элементов для крепления маркера и его движения вверх/вниз
        плата - 3D модель PCB_платы
        поле - полигон ЕТЗ
        Сборка - сборка всего ЕТЗ и наглядное предстовления его ввида (открывается файл только в Компас 3D)
        Электроника - Сборник моделей всех электронных компонентов и их держателей которые используется в ЕТЗ
